# 변경 이력

이 문서는 배포 버전의 사용자 관점 변경 사항을 기록한다.

## 3.4.0 - 2026-07-23

- Python FastAPI와 Flask, JSP, 일반 SQL 및 PL/SQL, SAP ABAP와 CDS 및 HANA 소스의 인덱싱과 연관 관계 분석 추가
- Rust, PHP, Ruby, Go 참조 및 안전 검증된 사내 전용 소스 확장자 설정 지원
- 다중 점 파일명의 업무 식별자 손실, SQL 더미 객체 과연결, SQL 확장자 분류 불일치와 민감 파일 판정 오류 수정
- 코드 펜스와 설명이 감싼 단일 JSON, 자연스러운 문장 종결, 중복 항목과 선택 메타데이터의 안전한 자동 정리
- 문체와 절차 수 차이 및 서술문 예시 값은 품질 검토 대상으로 전환하고 근거 없는 실행 대상 ID와 필수 JSON 구조는 계속 차단
- 릴리즈 노트 응답도 코드 펜스와 앞뒤 설명에서 단일 JSON 객체를 안전하게 추출
- 품질 기준을 충족한 초안의 불필요한 AI 재검토 생략과 교정 프롬프트 크기 제한
- provider User-Agent 버전 연동 및 사내용 EXE의 선택 Qwen provider 인증 설정 검증 수정
- 백신·EDR 환경에서 one-file `_MEI` 정리 경고가 반복될 때 사용할 공개·사내용 폴더형 배포 빌드 추가

## 3.3.4 - 2026-07-23

- PyInstaller 6.21.0 고정 및 UPX 비활성화로 사내 백신 환경 호환성 개선

## 3.3.3 - 2026-07-23

- 글머리표, 줄바꿈, 연속 공백과 마지막 마침표의 안전한 자동 정리
- 릴리즈 노트 뷰 닫기 버튼 폰트 크기 통일

## 3.3.2 - 2026-07-23

- CLI 시작 화면과 Windows EXE 파일 속성의 제품 버전 연동
- 제품 버전과 프롬프트 번들 버전의 동기화 검사 도입
- 릴리즈 노트 JSON 외곽 중괄호의 단순 중복·누락 자동 복구
